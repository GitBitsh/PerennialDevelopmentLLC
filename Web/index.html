Git Cheatsheet Configuration git config --global user.name "Your Name": Set your username globally. git config --global user.email "youremail@example.com": Set your email globally. Creating and Cloning Repositories git init: Initialize a new Git repository. git clone : Clone an existing repository to your local machine. Basic Workflow git add : Stage changes for commit. git commit -m "Commit message": Commit staged changes with a descriptive message. git push: Push commits to a remote repository. git pull: Fetch and merge changes from a remote repository. Branches git branch: List all branches in the repository. git branch : Create a new branch. git checkout : Switch to a different branch. git merge : Merge changes from a branch into the current branch. git branch -d : Delete a branch. Inspecting Changes git status: Show the current status of the repository. git log: Display the commit history. git diff: Show differences between the working directory and the staging area. Collaboration git remote add origin : Add a remote repository. git push -u origin : Push the branch to a remote repository. git pull origin : Pull changes from a remote repository. Undoing Changes git reset : Unstage changes for a file. git checkout : Discard changes in a file (can be used with a commit hash to restore a previous state).
